# 方案二——数据预处理

## src和tgt的数据格式
原始数据处理成的src和tgt的格式为：
```
src.txt                            tgt.txt
1. title content                   para1
xxxxxx                             para2
1.1 title content                  para1
xxxxxx                             para2
xxxxxx                             para3

2. titile content                  para1
2.1 titile content                 para2

3. titile content                  para1
......
```

Note:实际的src的数据是不会带标号的，这里只是为了形象说明
